如何打包项目文件夹
====================================================================

方法1: 使用Windows自带压缩

1. 在资源管理器中,选中整个项目文件夹
2. 右键点击
3. 选择 "发送到" → "压缩(zipped)文件夹"
4. 会生成: 20260311175444.zip

方法2: 使用WinRAR

1. 右键点击项目文件夹
2. 选择 "添加到压缩文件..."
3. 选择ZIP格式
4. 点击"确定"

方法3: 使用7-Zip

1. 右键点击项目文件夹
2. 选择 "7-Zip" → "添加到压缩包"
3. 选择ZIP格式
4. 点击"确定"

现在可以做什么?
====================================================================

选项1: 使用在线构建服务 (推荐)
1. 将项目文件夹打包为ZIP
2. 访问: https://appetize.io
3. 上传ZIP文件
4. 等待5-10分钟
5. 下载APK

选项2: 推送到GitHub
1. 安装Git (https://git-scm.com/download/win)
2. 打开命令提示符
3. 执行推送命令
4. 等待GitHub Actions构建
5. 下载APK

选项3: 安装Android Studio
1. 下载Android Studio
2. 打开项目
3. Build → Build APK(s)
4. 等待构建完成
5. 获得APK
